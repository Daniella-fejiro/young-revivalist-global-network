# Young Revivalist Global Network (YRGN) Website

## Overview

The Young Revivalist Global Network (YRGN) website is a modern, responsive digital platform designed to represent the vision, mission, and impact of YRGN. Built as an online ministry hub, the platform serves as a central space where individuals can learn about the ministry, connect with the community, access resources, and become part of a growing global movement of revivalists.

YRGN is a Spirit-led community committed to raising a generation of believers who are deeply rooted in Christ, empowered by the Holy Spirit, and equipped to influence their world for the Kingdom of God. This website reflects that mission by combining thoughtful design, engaging user experience, and accessible information.

---

## Purpose

This platform was created to:

* Present the vision, mission, and identity of YRGN.
* Introduce visitors to the ministry's history, leadership, and core values.
* Provide opportunities for individuals to join the movement.
* Offer clear channels for communication and community engagement.
* Showcase YRGN's growing presence across multiple locations.
* Serve as a digital gateway for revival, discipleship, and transformation.

---

## Installation

### Clone the repository

```bash
git clone <repository-url>
cd yrgn-website
```

### Install dependencies

```bash
npm install
```

### Start development server

```bash
npm run dev
```

---

## Build for Production

```bash
npm run build
```

---

## Deployment

This project is optimized for deployment on Vercel.

### Deploy with Vercel

```bash
npm install -g vercel
vercel
```

Or connect your GitHub repository directly through the Vercel dashboard for automatic deployments.
